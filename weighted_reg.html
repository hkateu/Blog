REGRESSION USING WEIGHTED LEAST SQUARES

This tutorial will teach you how to apply weighted least squares in Statsmodels.

We shall be using our food expenditure dataset

Importing the necessary libraries

import pandas as pd

from pandas import DataFrame, Series

import statsmodels.api as sm

We shall use 1/income as our weight for this example

Using Statsmodels Weighted Least Squares function WLS()

mod_wls = sm.WLS(foodexp, sm.add_constant(income), weights=1/income)

res_wls = mod_wls.fit()

print(res_wls.summary())
